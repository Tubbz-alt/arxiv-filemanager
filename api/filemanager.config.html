filemanager.config module¶

Flask configuration.

filemanager.config.KUBE_TOKEN = 'fookubetoken'¶

Service account token for authenticating with Vault. May be a file path.

filemanager.config.NAMESPACE = None¶

Namespace in which this service is deployed; to qualify keys for secrets.

filemanager.config.STORAGE_BACKEND = 'simple'¶

Name of the storage backend to use. See services.storage.

filemanager.config.VAULT_CERT = None¶

Path to CA certificate for TLS verification when talking to Vault.

filemanager.config.VAULT_ENABLED = False¶

Enable/disable secret retrieval from Vault.

filemanager.config.VAULT_HOST = 'foovaulthost'¶

Vault hostname/address.

filemanager.config.VAULT_PORT = '1234'¶

Vault API port.

filemanager.config.VAULT_REQUESTS = [{'type': 'generic', 'name': 'JWT_SECRET', 'mount_point': 'secret-None/', 'path': 'jwt', 'key': 'jwt-secret', 'minimum_ttl': 60}, {'type': 'database', 'engine': 'mysql+mysqldb', 'host': 'localhost', 'database': 'filemanager', 'params': 'charset=utf8mb4', 'port': '3306', 'name': 'SQLALCHEMY_DATABASE_URI', 'mount_point': 'database-None/', 'role': 'filemanager-write'}]¶

Requests for Vault secrets.

filemanager.config.VAULT_ROLE = 'filemanager'¶

Vault role linked to this application’s service account.

filemanager.config.VAULT_SCHEME = 'https'¶

Default is https.
