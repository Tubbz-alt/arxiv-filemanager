filemanager.services.database.translate module¶

Conversion of domain objects to/from native structs for persistence.

These are defined separately from the serialization functions that we use to prepare workspace data for API responses. In the API case, we are only exposing a subset of the information, and the naming/structure of the response documents in somewhat different from our internal representation. In contrast, the goal here is fidelity.
