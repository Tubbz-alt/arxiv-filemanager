get_single_file()[source]¶

Return File object for single-file submission.

This routine is intended for submission that are composed of a single content file.

Single file can’t be type ‘ancillary’. Single ancillary file is invalid submission and generates an error.

Returns

Single file. Returns None when submission has more than one file.

Return type

UploadedFile or None
