filemanager.controllers.source_log.check_upload_source_log_exists(upload_id)[source]¶

Determine if source log associated with upload workspace exists.

Note: This routine currently retrieves the source log for active upload workspaces. Technically, the upload source log is available for a ‘deleted’ workspace, since we stash this away before we actually delete the workspace. The justification to save is because the upload source log contains useful information that the admins sometime desire after a submission has been published and the associated workspace deleted.
